## Tools and Languages

- HTML
- SCSS
- JS
- [Vite](https://vitejs.dev)
- [Bootstrap Icons](https://icons.getbootstrap.com)

## Process

Creating and running the vite application using `npm`.

```sh
npm create vite@latest -- --template vanilla app

cd app

npm install

npm run dev
```
